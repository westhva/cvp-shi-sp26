⚙️ Activity Log Setup Guide

SHI × Canva SMB Q2 2026 Blitz  ·  One-time setup — takes about 15 minutes

What you’re setting up: A Google Form that reps use to log their activity. When they submit, a Google Apps Script automatically parses the accounts, enforces the 5-call/5-email cap, calculates points, and updates the leaderboard. You only need to do this once.

After setup, the rep workflow is: click Called/Emailed buttons in dashboard → click “Log Activity” → review pre-filled form → submit. Done.

1 Create the Google Sheet

  1. Go to sheets.google.com and create a new spreadsheet
  2. Name it something like SHI Blitz Activity Log
  3. You’ll set up the tabs in Step 4 — the Apps Script does that automatically

2 Create the Google Form

  1. Go to forms.google.com → click + Blank
  2. Name it: Canva x SHI — Activity Log
  3. Add a description: “Fill this in at the end of each outreach session. The accounts you’ve called and emailed will be pre-filled — just review, add any notes, and submit.”

Add these fields in this exact order:

#Question TextTypeRequired?
1Rep NameShort answerRequired
2Date Range
(e.g. “Apr 1” or “Apr 1–4”)		Short answerOptional
3Accounts Called
(pre-filled by dashboard)		Paragraph (long text)Optional
4Accounts Emailed
(pre-filled by dashboard)		Paragraph (long text)Optional
5Anything else to add?
Meetings, deals, corrections, notes		Paragraph (long text)Optional
📋 For meetings/deals, tell reps to write it like: “Meeting: Acme Corp, Deal Reg: Globex, Deal Booked: Initech” — the script will parse these automatically.

3 Link the Form to your Google Sheet

  1. In the Form editor, click Responses tab at the top
  2. Click the Google Sheets icon (“Link to Sheets”)
  3. Select “Select existing spreadsheet” and choose the sheet you made in Step 1
  4. Click Create — form responses will now flow into a “Form Responses” tab in your sheet
  5. Rename that tab to exactly: Submissions

4 Install the Apps Script

  1. In your Google Sheet, go to Extensions → Apps Script
  2. Delete any existing code in the editor
  3. Open the file activity_log_apps_script.gs (in the same folder as this guide), copy the entire contents, and paste it into the Apps Script editor
  4. Click Save (💾 icon or Ctrl+S)
  5. In the function dropdown at the top, select setup
  6. Click Run → follow the permissions prompt → click Allow
  7. You should see a popup: “Setup complete! Sheets created and form trigger installed.”
⚠️ Google will ask you to authorise the script to access your Sheet. This is normal — click Advanced then Go to [project name] (unsafe) if prompted. It’s your own script in your own account.

5 Get the Form pre-fill entry IDs

The dashboard needs the Form’s field IDs to pre-fill them. Here’s how to find them:

  1. In your Form editor, click the eye icon (“Preview”) to open a preview of the form
  2. In the preview, right-click the page and choose Inspect (or press F12)
  3. Use Ctrl+F to search for entry.
  4. You’ll see entries like entry.123456789 — one per field, in order
  5. Note down the entry ID for each field: Rep Name, Date Range, Accounts Called, Accounts Emailed
💡 Faster method: In your Form editor, click the 3-dot menu → Get pre-filled link. Type a test value in each field, click “Get link”, and the URL will show all entry IDs.

6 Get the Form’s submission URL

  1. In the Form editor, click Send (top right)
  2. Click the link icon
  3. Copy the URL — it looks like: https://docs.google.com/forms/d/e/XXXX/viewform

7 Update index.html with the Form URL and entry IDs

Open index.html in a text editor and find these lines near the top of the <script> section:

const FORM_BASE_URL = 'PLACEHOLDER_FORM_URL'; const ENTRY_REP = 'entry.PLACEHOLDER_REP'; const ENTRY_CALLED = 'entry.PLACEHOLDER_CALLED'; const ENTRY_EMAILED = 'entry.PLACEHOLDER_EMAILED';

Replace each placeholder with the real values you found in Steps 5 and 6:

const FORM_BASE_URL = 'https://docs.google.com/forms/d/e/YOUR_FORM_ID/viewform'; const ENTRY_REP = 'entry.1234567890'; // Rep Name field ID const ENTRY_CALLED = 'entry.2345678901'; // Accounts Called field ID const ENTRY_EMAILED = 'entry.3456789012'; // Accounts Emailed field ID

Save the file, then re-upload index.html to GitHub.

8 Connect the Leaderboard

  1. In your Google Sheet, go to the Leaderboard tab
  2. Go to File → Share → Publish to web
  3. Select Leaderboard sheet → Comma-separated values (.csv) → Publish
  4. Copy the published CSV URL
  5. Open leaderboard.html, find the SHEET_URL constant, and replace the PLACEHOLDER with this URL
  6. Re-upload leaderboard.html to GitHub

✓ You’re done!

After setup, the flow is fully automated:

  1. Rep clicks Called / Emailed buttons in dashboard as they go
  2. End of session → clicks Log Activity → pre-filled form opens
  3. Rep reviews, adds any meeting/deal notes, clicks Submit
  4. Apps Script parses the submission, enforces caps, calculates points, updates leaderboard
  5. Leaderboard auto-refreshes next time anyone loads it
🎮 To manually trigger a recalculation at any time (e.g. after editing a submission): open Apps Script → select recalcAll → Run.
